# Sweet Treats Bakery Website

📌 Project Overview

This is a simple and responsive bakery website created using HTML, CSS, and JavaScript. The website showcases bakery products like cakes, pastries, and cookies with a clean and attractive design.

# Features

* 🏠 Home page with bakery introduction
* 📋 Services page with product categories
* 🛒 Add to Cart system
* ➕➖ Quantity control for items
* 💰 Total price calculation
* 📲 WhatsApp order integration
* 📩 Contact form with success message

# Technologies Used

* HTML
* CSS
* JavaScript

# Purpose

The main goal of this project is to practice front-end development and build a real-world bakery website with basic e-commerce features.

# Future Improvements

* Online payment integration
* User login system
* Database for storing orders
